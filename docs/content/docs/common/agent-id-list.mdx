## Overview

The `AgentIDList` is a list of agent IDs, where each ID uniquely identifies an agent within
a group. It includes both the agent's unique ID and a Group ID (GID), allowing distinct identification
within their group and across different groups. The list offers methods to manage agent IDs,
such as adding, removing, and checking for specific agents.
