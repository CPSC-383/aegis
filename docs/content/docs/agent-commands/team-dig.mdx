## Overview

The `TEAM_DIG` command allows an agent to dig rubble.
If the rubble needs multiple agents to remove it, all 
required agents must be on the same cell and use `TEAM_DIG`
in the same round.

**Example:**

```py
self._agent.send(TEAM_DIG())
