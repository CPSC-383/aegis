## Overview

The `OBSERVE` command allows an agent to gather information about a specific cell in
the world. Observing a cell provides details about it (`CellInfo`) and any detected
life signals (`LifeSignals`).

## Params

<Attribute
  name="location"
  type="Location"
  description="The location to observe."
  required
/>

**Example:**

```py
# Observing cell at X: 10, Y: 15 
loc = create_location(10, 15)
self._agent.send(OBSERVE(loc))
```
