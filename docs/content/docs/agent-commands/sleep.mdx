---

## Overview

The `SLEEP` command allows an agent to sleep and recharge energy. For recharging to occur, 
the agent must be on a charging grid when the command is sent. Each time the agent sleeps, 
it gains 5 energy.

**Example:**

```py
self._agent.send(SLEEP())
```
