---
title: "Predict Result"
description: "Represents the result of an agent's prediction."
---

## Overview
The `PREDICT_RESULT` represents the outcome of an agent's prediction. It includes
the ID of the survivor associated with the prediction and whether the prediction was correct.

## Attributes

<Attribute
  name="surv_id"
  type="int"
  description="The id of the survivor (predictions are based on saved survivors, so this is the id of the survivor associated with the prediction)."
/>

<Attribute
  name="prediction_correct"
  type="bool"
  description="If the agent's prediction was correct or not."
/>
