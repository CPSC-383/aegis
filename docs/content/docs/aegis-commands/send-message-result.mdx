## Overview

The `SEND_MESSAGE_RESULT` represents a message received from another agent.
It includes the sender's ID, the list of intended recipients, and the message content.

## Attributes

<Attribute
  name="from_agent_id"
  type="AgentID"
  description="The AgentID of the agent sending the message."
/>

<Attribute
  name="agent_id_list"
  type="AgentIDList"
  description="The list of agents who are supposed to receive the message."
/>

<Attribute
  name="msg"
  type="str"
  description="The content of the message."
/>
