title: get_cardinal_directions
---

## Overview

Returns a list of all cardinal directions.

<Callout title="Static Method Usage" type="warn">

This method should be called on the `Direction` class, not on an instance.

Correct:

```py
Direction.get_cardinal_directions()
```

Wrong:

```py
dir = Direction.NORTH
dir.get_cardinal_directions()
```

</Callout>

## Signature

```py
get_cardinal_directions() -> list[Direction]
```

## Example

```py
cardinals = Direction.get_cardinal_directions()
print(cardinals)  # Output: [Direction.NORTH, Direction.EAST, Direction.SOUTH, Direction.WEST]
```
