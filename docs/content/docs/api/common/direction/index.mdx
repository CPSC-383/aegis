---
title: Direction 
description: Enum representing different directions, used for navigating the world.
---

## Overview

The `Direction` class provides a flexible and powerful interface for working with two-dimensional coordinates.
It defines a set of cardinal and intercardinal directions that can be used for navigating the world.

## Members

| Direction     | dx  | dy  | Description        |
|----------------|-----|-----|--------------------|
| `NORTH`        | 0   | 1   | Upward             |
| `NORTH_EAST`   | 1   | 1   | Upward-Right       |
| `EAST`         | 1   | 0   | Right              |
| `SOUTH_EAST`   | 1   | -1  | Downward-Right     |
| `SOUTH`        | 0   | -1  | Downward           |
| `SOUTH_WEST`   | -1  | -1  | Downward-Left      |
| `WEST`         | -1  | 0   | Left               |
| `NORTH_WEST`   | -1  | 1   | Upward-Left        |
| `CENTER`       | 0   | 0   | No movement        |

## Attributes

| Attribute | Type | Description                          |
|------------|------|--------------------------------------|
| `dx`       | `int`  | The change in the x-direction.       |
| `dy`       | `int`  | The change in the y-direction.       |
