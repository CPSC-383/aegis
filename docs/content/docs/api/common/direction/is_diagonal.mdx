title: is_diagonal
---

## Overview

Returns `True` if the direction is diagonal (`NORTH_EAST`, `SOUTH_EAST`, `NORTH_WEST`, `SOUTH_WEST`).

## Signature

```py
is_diagonal() -> bool
```

## Example

```py
print(Direction.NORTH_EAST.is_diagonal())  # Output: True
print(Direction.SOUTH.is_diagonal())  # Output: False
```
