---
title: get_movement_directions
---

## Overview

Returns all directions that represent movement, excluding `CENTER`.

<Callout title="Static Method Usage" type="warn">

This method should be called on the `Direction` class, not on an instance.

Correct:

```py
Direction.get_movement_directions()
```

Wrong:

```py
dir = Direction.NORTH
dir.get_movement_directions()
```

</Callout>

## Signature

```py
get_movement_directions() -> list[Direction]
```

## Example

```py
movements = Direction.get_movement_directions()
print(movements)  # Output: List of directions excluding CENTER
```
