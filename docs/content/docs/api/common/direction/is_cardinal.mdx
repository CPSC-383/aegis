---
title: is_cardinal
---

## Overview

Returns `True` if the direction is a cardinal direction (`NORTH`, `EAST`, `SOUTH`, `WEST`).

## Signature

```py
is_cardinal() -> bool
```

## Example

```py
print(Direction.EAST.is_cardinal())  # Output: True
print(Direction.NORTH_WEST.is_cardinal())  # Output: False
```
