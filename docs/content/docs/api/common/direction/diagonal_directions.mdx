---
title: get_diagonal_directions
---

## Overview

Returns a list of all diagonal directions.

<Callout title="Static Method Usage" type="warn">

This method should be called on the `Direction` class, not on an instance.

Correct:

```py
Direction.get_diagonal_directions()
```

Wrong:

```py
dir = Direction.NORTH
dir.get_diagonal_directions()
```

</Callout>

## Signature

```py
get_diagonal_directions() -> list[Direction]
```

## Example

```py
diagonals = Direction.get_diagonal_directions()
print(diagonals)  # Output: [Direction.NORTH_EAST, Direction.SOUTH_EAST, Direction.SOUTH_WEST, Direction.NORTH_WEST]
```
